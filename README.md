### Installation
<pre>
cd /home
git clone https://github.com/ehrmaks/Jookbob2-Blog.git
cd ./Jookbob2-Blog
</pre>

### Run
<pre>
cd /home/Jookbob2-Blog
docker-compose down
git pull
docker-compose up -d --build
</pre>
